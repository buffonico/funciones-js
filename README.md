# Desafio Funciones

En este proyecto se realizaron ejercicios de funciones. 

Los ejercicios 1_funcion.js y 2_arrow.js son transformaciones de funcion a expresion y arrows respectivamente.

## Link Acceso Pages y Github
    https://github.com/buffonico/funciones-js
    https://buffonico.github.io/funciones-js/


## Requerimientos.

- Ejercicio 1 - "1_funcion.js".
- Ejercicio 2 - "2_arrow.js".
- Ejercicio 3 Se pide que al hacer click, el boton cambie a color amarillo. Archivos "pintar.html y script.js".
- Ejercicio 4 Se pide que al hacer click en unos cuadros cambie de color negro. "4_colores.html y 4_colores.js".
- Ejercicio 5 Se pide que al apretar las teclas indicadas, se agregue el color del requerimiento. "keydown.html y keydown.js".


## Herramientas Utilizadas

- HTML5
- CSS3
- JavaScript
